ZUR SOFORTIGEN VERÖFFENTLICHUNG

Kerpen, Deutschland, den 31.03.2011. Pomcast.biz freut sich, nach mehreren Jahren der iOS Beratung, seine erste Applikation vorstellen zu können.

Mac OS X informiert Sie, wenn Ihre Festplatte voll ist, aber meistens ist es dann bereits zu spät um zu reagieren.

Sie würden gerne früh genug benachrichtigt werden, so dass Sie Ihre Dateien verschieben oder löschen können, um wieder genug Platz auf der Festplatte zu haben, damit Ihr Mac reibungslos läuft?

Disk Alarm wird Sie im Voraus informieren, und Sie sparen eine Menge Zeit!

Features:
- Disk Alarm informiert Sie, wenn Sie einen bestimmten Höhe der Speicherkapazität erreicht haben.
- Optional werden Sie mit einem Geräusch informiert, was sehr nützlich ist, wenn sie gerade nicht vor dem Bildschirm sitzen.
- Es läuft wie ein agent app - wenn das Einstellung Fenster geschlossen ist wird es Sie noch immer informieren.
- Optional bleibt Disk Alarm in Ihrer Menüleiste um schnellen Zugriff darauf zu haben und den freien Platz anzuzeigen.
- Wenn Sie den freien Platz nicht automatisch angezeigt haben wollen, können Sie dies auch manuell überprüfen
- Optional können sie Disk Alarm zu Ihren Login Items hinzufügen, so dass er immer läuft, sobald der Computer hochfährt.
- Es gibt die Möglichkeit, den überprüften Platz in Prozenten anstatt in MB/ GB zu sehen

Minimale Anforderungen:
- Mac OS X Version 10.6.6 oder höher
- Intel Mac

Nachdem vielen Kunden seit dem Beginn des SDK in 2008 bei iOS Projekten geholfen wurde, startet Pomcast.biz endlich seine erste Mac Applikation: "Disk Alarm".
Diese Applikation, die den Beginn einer Reihe von Mac und iOS Applikationen darstellt, überwacht Ihre Festplatte und meldet Ihnen, wenn eine gewisse Speicherkapazität erreicht wird, die zuvor von Ihnen festgelegt wurde.
