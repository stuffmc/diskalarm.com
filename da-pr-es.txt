PARA DIFUSIÓN INMEDIATAKerpen, Alemania - 4 de abril de 2001 - Pomcast.biz se enorgullece de presentar su primera aplicación Mac OS X, después de varios años de consultor iOS.Mac OS X le avisa cuando su disco duro está lleno pero, en general, ya es demasiado tarde para reaccionar.¿Le gustaría recibir una alerta con suficiente antelación para que pueda mover o borrar archivos, de forma que le permita tener suficiente espacio libre para que su Mac funcione sin problemas?¡Disk Alarm le avisará con suficiente antelación y le ahorrará mucho tiempo!Características:* Le avisa cuando llega a una cierta cantidad de espacio libre en disco.* Opcionalmente avisa con un sonido, muy útil cuando usted no está delante de la pantalla del ordenador.* Se ejecuta como una aplicación oculta: aunque cierre la ventana de configuración, el programa le avisará cuando proceda.* Opcionalmente se queda en la barra de menús permitiendo un acceso rápido y una visualización del espacio libre del disco.* Comprueba el espacio en disco de forma manual si usted no quiere hacerlo de forma automática.* Añadido (opcionalmente) a los items de arranque, la aplicación se ejecutará cuando se inicia el equipo.* Próximamente en la versión 1.0.1: Capacidad de ver el “espacio marcado” en porcentajes en vez de MB/GB.Requisitos mínimos:* Mac OS X versión 10.6.6 o superior* Intel MacEstaríamos encantados de proporcionarle un Código Promocional para que pueda revisar la aplicación.
